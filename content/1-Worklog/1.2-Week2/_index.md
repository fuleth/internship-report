### Week 2 Achievements:

* Gained a solid understanding of Amazon VPC architecture, including Subnets, Elastic Network Interfaces (ENIs), Elastic IP Addresses, Internet Gateways, and NAT Gateways.
* Learned key VPC security concepts such as Security Groups and Network ACLs, and explored Multi-VPC connectivity using VPC Peering and Transit Gateway.
* Understood hybrid networking solutions with AWS Site-to-Site VPN and AWS Direct Connect, as well as the fundamentals of Elastic Load Balancing.
* Completed hands-on labs for Amazon VPC and AWS Site-to-Site VPN to reinforce networking concepts through practical experience.
* Practiced configuring Route 53, VPC Peering, and Transit Gateway, gaining experience in DNS management and connecting multiple VPCs securely and efficiently.
