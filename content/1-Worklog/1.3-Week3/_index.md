### Week 3 Objectives:

* Understand AWS compute services, with a primary focus on Amazon EC2 and related storage options.
* Learn how to deploy, manage, and scale EC2 instances using AWS best practices.
* Explore monitoring, migration, and resource management services such as CloudWatch, AWS MGN, and Resource Groups.
* Gain practical experience by completing hands-on labs for EC2, EBS, EFS, Lightsail, and AWS MGN.

### Week 3 Achievements:

* Built a solid understanding of Amazon EC2, including instance types, Amazon Machine Images (AMIs), Key Pairs, User Data, Instance Metadata, and storage options such as Amazon EBS, Instance Store, Amazon EFS, and Amazon FSx.
* Learned how to improve application availability and scalability using EC2 Auto Scaling, and explored AWS Lightsail as a simplified cloud computing solution.
* Gained knowledge of AWS CloudWatch for monitoring resources, AWS Application Migration Service (MGN) for workload migration, and Tag & Resource Groups for efficient resource organization and management.
* Successfully completed hands-on labs involving Amazon EC2, Amazon EBS, Amazon EFS, AWS Lightsail, and AWS MGN, reinforcing theoretical concepts through practical implementation.
* Developed practical skills in provisioning compute resources, managing storage, monitoring workloads, and performing basic migration tasks in the AWS Cloud.
