### Mục tiêu tuần 4:

* Hiểu các dịch vụ lưu trữ của AWS, tập trung vào Amazon S3 và các tính năng cốt lõi của dịch vụ.
* Học cách tối ưu chi phí lưu trữ và quản lý dữ liệu bằng Amazon S3 Storage Classes và Lifecycle Policies.
* Tìm hiểu các giải pháp lưu trữ Hybrid Cloud, di chuyển dữ liệu quy mô lớn và khôi phục sau thảm họa (Disaster Recovery) trên AWS.
* Tích lũy kinh nghiệm thực tế thông qua các bài lab về AWS Backup, VM Import/Export, Storage Gateway và các dịch vụ di chuyển dữ liệu.

### Kết quả đạt được tuần 4:

* Nắm vững kiến thức về Amazon S3, bao gồm các khái niệm Object Storage, quản lý Bucket và Object, REST API, kiến trúc lưu trữ cũng như các tính năng quan trọng như Access Points, Static Website Hosting, CORS, Versioning và VPC Endpoints.
* Học cách tối ưu chi phí lưu trữ bằng cách sử dụng Amazon S3 Storage Classes và Object Lifecycle Management.
* LTìm hiểu các dịch vụ thuộc AWS Snow Family (Snowball, Snowball Edge và Snowmobile) phục vụ di chuyển dữ liệu quy mô lớn, cùng với AWS Storage Gateway (File Gateway, Volume Gateway và Tape Gateway) để xây dựng giải pháp lưu trữ Hybrid Cloud.
* Hiểu các khái niệm về khôi phục sau thảm họa (Disaster Recovery), bao gồm Recovery Time Objective (RTO) và Recovery Point Objective (RPO), cũng như vai trò của chúng trong việc thiết kế hệ thống có tính sẵn sàng và khả năng phục hồi cao.
* Hoàn thành các bài lab thực hành về AWS Backup, VM Import/Export, File Storage Gateway và AWS Application Migration Service (MGN), qua đó nâng cao kỹ năng thực tế trong sao lưu dữ liệu, di chuyển hệ thống và triển khai giải pháp lưu trữ Hybrid Cloud. Lab 25 không thể hoàn thành do nội dung đã lỗi thời.
