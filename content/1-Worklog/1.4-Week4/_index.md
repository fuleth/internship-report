### Week 4 Objectives:

* Understand AWS storage services, with a primary focus on Amazon S3 and its core features.
* Learn how to optimize storage costs and data management using S3 Storage Classes and Lifecycle policies.
* Explore hybrid cloud storage, large-scale data migration, and disaster recovery solutions on AWS.
* Gain hands-on experience with AWS Backup, VM Import/Export, Storage Gateway, and migration services through practical labs.

### Week 4 Achievements:

* Developed a solid understanding of Amazon S3, including object storage concepts, bucket and object management, REST API, storage architecture, and key features such as Access Points, Static Website Hosting, CORS, Versioning, and VPC Endpoints.
* Learned how to optimize storage costs using Amazon S3 Storage Classes and Object Lifecycle Management.
* Explored AWS Snow Family services (Snowball, Snowball Edge, and Snowmobile) for large-scale data migration, as well as AWS Storage Gateway solutions (File Gateway, Volume Gateway, and Tape Gateway) for hybrid cloud storage.
* Gained knowledge of disaster recovery concepts, including Recovery Time Objective (RTO) and Recovery Point Objective (RPO), and their role in designing resilient systems.
* Successfully completed hands-on labs for AWS Backup, VM Import/Export, File Storage Gateway, and AWS Application Migration Service (MGN), strengthening practical skills in backup, migration, and hybrid storage deployment. Lab 25 could not be completed because it was outdated.
