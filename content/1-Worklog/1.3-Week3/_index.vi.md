### Mục tiêu tuần 3:

* Hiểu các dịch vụ điện toán (Compute) của AWS, tập trung vào Amazon EC2 và các dịch vụ lưu trữ liên quan.
* Học cách triển khai, quản lý và mở rộng các EC2 Instance theo các phương pháp thực hành tốt nhất của AWS.
* Tìm hiểu các dịch vụ giám sát, di chuyển hệ thống và quản lý tài nguyên như Amazon CloudWatch, AWS Application Migration Service (MGN) và Resource Groups.
* Tích lũy kinh nghiệm thực tế thông qua các bài lab về Amazon EC2, Amazon EBS, Amazon EFS, AWS Lightsail và AWS MGN.

### Kết quả đạt được tuần 3:

* Nắm vững kiến thức về Amazon EC2, bao gồm các loại Instance, Amazon Machine Image (AMI), Key Pair, User Data, Instance Metadata và các tùy chọn lưu trữ như Amazon EBS, Instance Store, Amazon EFS và Amazon FSx.
* Học cách nâng cao tính sẵn sàng và khả năng mở rộng của ứng dụng bằng EC2 Auto Scaling, đồng thời tìm hiểu AWS Lightsail như một giải pháp điện toán đám mây đơn giản hóa.
* Tìm hiểu Amazon CloudWatch để giám sát tài nguyên, AWS Application Migration Service (MGN) để di chuyển khối lượng công việc lên AWS, cùng Tag và Resource Groups để tổ chức và quản lý tài nguyên hiệu quả.
* Hoàn thành các bài lab thực hành về Amazon EC2, Amazon EBS, Amazon EFS, AWS Lightsail và AWS MGN, giúp củng cố kiến thức lý thuyết thông qua trải nghiệm thực tế.
* Phát triển kỹ năng thực hành trong việc triển khai tài nguyên điện toán, quản lý lưu trữ, giám sát hệ thống và thực hiện các tác vụ di chuyển cơ bản trên nền tảng AWS Cloud.
