### Week 6 Objectives:

* Build a strong foundation in database concepts and understand their role in modern applications.
* Learn AWS managed database services, including Amazon RDS, Amazon Aurora, Amazon Redshift, and Amazon ElastiCache.
* Explore database migration solutions using AWS Database Migration Service (DMS) and AWS Schema Conversion Tool (SCT).
* Gain hands-on experience deploying and migrating databases through practical AWS labs.

### Week 6 Achievements:

* Reviewed fundamental database concepts, including database architecture, core RDBMS components, transaction management, performance optimization, data models, and the differences between OLTP and OLAP workloads.
* Developed a solid understanding of Amazon RDS and Amazon Aurora, including their architecture, key features, high availability capabilities, and the advanced features that make Aurora a high-performance managed relational database.
* Learned how Amazon Redshift supports large-scale data warehousing through massively parallel processing (MPP) and columnar storage, and explored Amazon ElastiCache for improving application performance with Redis and Memcached.
* Explored AWS Database Migration Service (DMS) and AWS Schema Conversion Tool (SCT) to understand database migration strategies and schema conversion between different database engines.
* Successfully completed hands-on labs for Amazon RDS and Schema Conversion & Database Migration, gaining practical experience in deploying managed databases and performing database migration on AWS.
